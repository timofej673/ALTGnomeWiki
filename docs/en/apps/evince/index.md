# Evince

Evince is the official utility for the GNOME desktop environment, which supports documents in the following formats: PDF, PS, EPS, XPS, DjVu, TIFF, DVI (with SyncTeX) and Comic Book Archive (CBR, CBT, CBZ, CB7).
