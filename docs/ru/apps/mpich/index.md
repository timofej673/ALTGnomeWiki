# MPICH

MPICH — высокопроизводительная реализация MPI для параллельных вычислений.

## Установка из репозитория

**MPICH** можно установить любым привычным и удобным способом:

**Установка через терминал**

::: code-group

```shell-vue[apt-get]
su -
apt-get update
apt-get install mpich
```

```shell-vue[epm]
epm -i mpich
```

:::
